<tr><td align=center><h1 style='color:#1A5CC8'>Adjacent Bit Counts</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 2000/1000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 32768/32768 K (Java/Others)<br>Total Submission(s): 508&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 423<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>For a string of n bits x<sub>1</sub>, x<sub>2</sub>, x<sub>3</sub>, …, x<sub>n</sub>, the adjacent bit count of the string (<b>AdjBC(x)</b>) is given by<br><br><b>x<sub>1</sub>*x<sub>2</sub> + x<sub>2</sub>*x<sub>3</sub> + x<sub>3</sub>*x<sub>4</sub> + … + x<sub>n-1</sub>*x<sub>n</sub></b><br><br>which counts the number of times a 1 bit is adjacent to another 1 bit. For example:<br><br>AdjBC(011101101) = 3<br>AdjBC(111101101) = 4<br>AdjBC(010101010) = 0<br><br>Write a program which takes as input integers n and k and returns the number of bit strings x of n bits (out of 2<sup>n</sup>) that satisfy AdjBC(x) = k. For example, for 5 bit strings, there are 6 ways of getting<br>AdjBC(x) = 2:<br><br>11100, 01110, 00111, 10111, 11101, 11011</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>The first line of input contains a single integer P, (1 ≤ P ≤ 1000), which is the number of data sets that follow. Each data set is a single line that contains the data set number, followed by a space, followed by a decimal integer giving the number (n) of bits in the bit strings, followed by a single space, followed by a decimal integer (k) giving the desired adjacent bit count. The number of bits (n) will not be greater than 100 and the parameters n and k will be chosen so that the result will fit in a signed 32-bit integer.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For each data set there is one line of output. It contains the data set number followed by a single space, followed by the number of n-bit strings with adjacent bit count equal to k.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">10
1 5 2
2 20 8
3 30 17
4 40 24
5 50 37
6 60 52
7 70 59
8 80 73
9 90 84
10 100 90</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">1 6
2 63426
3 1861225

7 22937308
