<tr><td align=center><h1 style='color:#1A5CC8'>Cat</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 2000/1000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 32768/32768 K (Java/Others)<br>Total Submission(s): 564&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 146<br><font color=red>Special Judge</font></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>There is a cat, cat likes to sleep. <br>If he sleeps, he sleeps continuously no less than A hours.<br>For some strange reason, the cat can not stay awake continuously more than B hours.<br>The cat is lazy, it could sleep all the time, <br>but sometimes interesting events occur(such as cat food, TV show, etc) . <br>The cat loves these events very much.<br>So, Please help the cat plan their day so as not to miss any interesting events. <br>Every day the cat wants to live on the same schedule.<br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>The first line of the input file contains two integers A and B (1 &lt;= A, B &lt;= 24). <br>The second line of the input file contains the number n, the number of interesting events (1 &lt;= n &lt;= 20). <br>Following n rows describe the interesting events. <br>Each event is described line of the form hh:mm-hh:mm, which specifies<br>the time period during which it occurs. Time varies from 00:00 to 23:59. <br>No two interesting events will overlap.<br> If the event is completed earlier than the beginning, This means that it captures midnight.<br>The event is considered to be occupying the whole minute, <br>when it begins and the moment when it ends (event 12:00-13:59 lasted exactly 120 minutes). Start time and time end of the event are different.<br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>If the cat can organize their day so that during all the interesting events not sleep, print to output file Yes.<br><br>On the second line Bring k - how many times a day cat should go to bed. <br>In the following k rows Bring out the intervals in which the cat sleeps in the same format, in which interesting events are set in the input file. If making a few, display any.<br><br>If the cat can not organize their day desired way, print to the output file No.<br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">12 12
1
23:00-01:00
3 4
3
07:00-08:00
11:00-11:09
19:00-19:59</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">Yes
1
01:07-22:13
