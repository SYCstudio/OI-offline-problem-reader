<tr><td align=center><h1 style='color:#1A5CC8'>Fraction</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 2000/1000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 65536/65536 K (Java/Others)<br>Total Submission(s): 663&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 233<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>   Given a number n, and a geometric progression a<sub>i</sub> = b * q<sub>i</sub>, i ≥ 0, what is the fraction of the elements of that progression with decimal notation that has the decimal notation of n as prefix ?<br><br>   More formally, if c<sub>i</sub> out of the first i elements of the progression start with n in decimal notation, you need to find the limit<img style='max-width:100%;' src=../../data/images/C537-1010-1.jpg>. It is guaranteed that the limit always exists.<br><br>   For example, n = 7, b = 1, q = 2. About 5.799% of all powers of two start with 7. (the smallest one is 2<sup>46</sup> = 70368744177664)</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>   The first line of the input is T (1 ≤ T ≤ 100), which stands for the number of test cases you need to solve.<br><br>   Each case contains three integers n,b and q. (1 ≤ n, b, q ≤ 1000)</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>   For each test case, print a line “Case #t: ”(without quotes, t means the index of the test case) at the beginning. Then output one floating number – the sought fraction. Round your answer to the 5th decimal place.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">2
7 1 2
1 1 1</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">Case #1: 0.05799
