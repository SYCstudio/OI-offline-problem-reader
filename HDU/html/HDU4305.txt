<tr><td align=center><h1 style='color:#1A5CC8'>Lightning</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 4000/2000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 32768/32768 K (Java/Others)<br>Total Submission(s): 2316&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 844<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>There are N robots standing on the ground (Don't know why. Don't know how). <br><center><img style='max-width:100%;' src=../../data/images/C398-1006-1.jpg></center><br>Suddenly the sky turns into gray, and lightning storm comes! Unfortunately, one of the robots is stuck by the lightning!<br><center><img style='max-width:100%;' src=../../data/images/C398-1006-2.jpg></center><br>So it becomes overladen. Once a robot becomes overladen, it will spread lightning to the near one.<br><center><img style='max-width:100%;' src=../../data/images/C398-1006-3.jpg></center><br><br>The spreading happens when: <br>&nbsp;&nbsp;Robot A is overladen but robot B not.<br>&nbsp;&nbsp;The Distance between robot A and robot B is no longer than R.<br>&nbsp;&nbsp;No other robots stand in a line between them.<br>In this condition, robot B becomes overladen. <br><br>We assume that no two spreading happens at a same time and no two robots stand at a same position. <br><br><center><img style='max-width:100%;' src=../../data/images/C398-1006-4.jpg></center><br>The problem is: How many kind of lightning shape if all robots is overladen? The answer can be very large so we output the answer modulo 10007. If some of the robots cannot be overladen, just output -1. <br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>There are several cases.<br>The first line is an integer T (T &lt; = 20), indicate the test cases.<br>For each case, the first line contains integer N ( 1 &lt; = N &lt; = 300 ) and R ( 0 &lt; = R &lt; = 20000 ), indicate there stand N robots; following N lines, each contains two integers ( x, y ) ( -10000 &lt; = x, y &lt; = 10000 ), indicate the position of the robot. <br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>One line for each case contains the answer.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">3
3 2
-1 0
0 1
1 0
3 2
-1 0
0 0
1 0
3 1
-1 0
0 1
1 0</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">3
