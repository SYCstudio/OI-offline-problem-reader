<tr><td align=center><h1 style='color:#1A5CC8'>It's All In The Mind</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 2000/1000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 65536/65536 K (Java/Others)<br>Total Submission(s): 2475&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 1061<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>Professor Zhang has a number sequence $a_1,a_2,...,a_n$. However, the sequence is not complete and some elements are missing. Fortunately, Professor Zhang remembers some properties of the sequence:<br><br>1. For every $i \in \{1,2,...,n\}$, $0 \le a_i \le 100$.<br>2. The sequence is non-increasing, i.e. $a_1 \ge a_2 \ge ... \ge a_n$.<br>3. The sum of all elements in the sequence is not zero.<br><br>Professor Zhang wants to know the maximum value of $\frac{a_1+a_2}{\sum_{i=1}^{n}{a_i}}$ among all the possible sequences.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>There are multiple test cases. The first line of input contains an integer $T$, indicating the number of test cases. For each test case:<br><br>The first contains two integers $n$ and $m$ $(2 \le n \le 100, 0 \le m \le n)$ -- the length of the sequence and the number of known elements.<br><br>In the next $m$ lines, each contains two integers $x_i$ and $y_i$ $(1 \le x_i \le n, 0 \le y_i \le 100, x_i &lt; x_{i+1}, y_i \ge y_{i+1})$, indicating that $a_{x_i} = y_i$.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For each test case, output the answer as an irreducible fraction &quot;$p$/$q$&quot;, where $p$, $q$ are integers, $q &gt; 0$.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">2
2 0
3 1
3 1</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">1/1
