<tr><td align=center><h1 style='color:#1A5CC8'>Bomberman</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 3000/3000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 65535/32768 K (Java/Others)<br>Total Submission(s): 89&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 20<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>The story begins with bomberman growing bored of making bombs in an underground factory of the Bungeling Empire. After hearing a rumor that robots reaching his surface become human, he decides to escape. When he does, he transform into an organic human being. To distinguish him from other bombermen, our main character is given the name White Bomber. However another bomberman, known as Black Bomber is an enemy due to a programming error, trying to destroy the entire map.<br>A two dimensional lattice with a coefficient $H~(4\le H\le 100)$ describes the map. All pairs of integers $(x,y)$ satisfying $x+y=H$ or $x+y=H+1$ but $(0,H+1),(0,-H-1),(H+1,0)$ and $(-H-1,0)$ make up the range. Were a bomb deployed at a grid $(x,y)$ by Black Bomber, all grids around it would disappear, including $(x-1,y-1)$, $(x-1,y)$, $(x-1,y+1)$, $(x,y-1)$, $(x,y)$, $(x,y+1)$, $(x+1,y-1)$, $(x+1,y)$ and $(x+1,y+1)$ if exist.<br><center><img style='max-width:100%;' src=../../../data/images/C778-1002-1.jpg></center><br>The location of the bomb is an uniformly random every time when Black Bomber deploys it, but bombs can not be placed at disappeared grids. Black Bomber deploys a bomb and ignites it, then deploys anothor bomb and ignites it again. He stops if the whole map has been destroyed. Help White Bomber calculate the expected number of bombs which would be deployed by Black Bomber. Furthermore, if White Bomber has discovered the first grid which Black Bomber would select, what is the expected number of bombs that would be deployed.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>The first line is an integer $T~(1\le 100\le T)$ which is the number of test cases.<br>For each test case, there is a line containing an integer $H~(4\le H\le 100)$.<br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For each test case, output two lines.<br>The first contains the the expected number of bombs deployed with the precision of $6$ digits.<br>The second line contains expected numbers if the first grid has been confirmed.<br>Output $8H$ floats numbers with $6$ places in a line corresponding to all grids in the clockwise direction and the first one indicates the situation of the top position $(0,H)$.<br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">2
