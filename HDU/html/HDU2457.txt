<tr><td align=center><h1 style='color:#1A5CC8'>DNA repair</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 5000/2000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 32768/32768 K (Java/Others)<br>Total Submission(s): 3243&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 1714<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>Biologists finally invent techniques of repairing DNA that contains segments causing kinds of inherited diseases. For the sake of simplicity, a DNA is represented as a string containing characters 'A', 'G' , 'C' and 'T'. The repairing techniques are simply to change some characters to eliminate all segments causing diseases. For example, we can repair a DNA &quot;AAGCAG&quot; to &quot;AGGCAC&quot; to eliminate the initial causing disease segments &quot;AAG&quot;, &quot;AGC&quot; and &quot;CAG&quot; by changing two characters. Note that the repaired DNA can still contain only characters 'A', 'G', 'C' and 'T'.<br><br>You are to help the biologists to repair a DNA by changing least number of characters.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>The input consists of multiple test cases. Each test case starts with a line containing one integers N (1 ≤ N ≤ 50), which is the number of DNA segments causing inherited diseases.<br>The following N lines gives N non-empty strings of length not greater than 20 containing only characters in &quot;AGCT&quot;, which are the DNA segments causing inherited disease.<br>The last line of the test case is a non-empty string of length not greater than 1000 containing only characters in &quot;AGCT&quot;, which is the DNA to be repaired.<br><br>The last test case is followed by a line containing one zeros.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For each test case, print a line containing the test case number( beginning with 1) followed by the<br>number of characters which need to be changed. If it's impossible to repair the given DNA, print -1.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">2
AAA
AAG
AAAG    
2
A
TG
TGAATG
4
A
G
C
T
AGT
0</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">Case 1: 1
Case 2: 4
Case 3: -1</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Source</div> <div class=panel_content> <a href="/search.php?field=problem&key=2008+Asia+Hefei+Regional+Contest+Online+by+USTC&source=1&searchmode=source"> 2008 Asia Hefei Regional Contest Online by USTC </a> </div> <div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Recommend</div> <div class=panel_content>teddy</div><div class=panel_bottom>&nbsp;</div><br><center style='font-size:15px;font-family:Arial;font-weight:bold;color:#1A5CC8'><a href='statistic.php?pid=2457'>Statistic</a>&nbsp;|&nbsp;<a href='submit.php?pid=2457'>Submit</a>&nbsp;|&nbsp;<a href="./discuss/problem/list.php?problemid=2457">Discuss</a> | <a href='note/note.php?pid=2457'>Note</a><br></td></tr><tr>
