<tr><td align=center><h1 style='color:#1A5CC8'>Colorful Tree</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 24000/12000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 131072/131072 K (Java/Others)<br>Total Submission(s): 556&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 112<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>As we all know, frogs live on trees and have different colors.<br><br>$N$ frogs are living on a tree. The tree consists of $N$ nodes with node $1$ as the root, each frog occupies a node.<br><br>Frogs have different colors, and can change colors as they like. On each day, all the frogs living on a certain sub-tree will change its color. The root of the sub-tree, and the color they change to, is given to the frog king.<br><br>As the frog king, sometimes he may wonder, how many different colors of frog are there in a certain sub-tree? It turns to you to solve the problem for the king.<br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>First line contains an integer $T$, which indicates the number of test cases.<br><br>Every test case begins with an integers $N$, which is the numbers of nodes in the tree.<br><br>The following $N - 1$ lines describe the edges of the tree, and every line is formatted as '$u\ v$', which indicates there is a edge between node $u$ and node $v$.<br><br>The next line contains $N$ intergers, $c_1$, $c_2$, $\cdots$, $c_N$, and $c_i$ is the initial color of the frog living at node $i$.<br><br>Then a number $M$ follows, which indicates the number of queries, and following $M$ lines describe the quries as format bellow.<br><br><center><img style='max-width:100%;' src=../../data/images/C655-1003-1.png></center><br><br>$\cdot$ $1 \leq T \leq 100$.<br><br>$\cdot$ For 85% data, $1 \leq N, M \leq 1000$.<br><br>$\cdot$ for 100% data, $1 \leq N, M \leq 10^5$.<br><br>$\cdot$ for every node, $1 \leq c_i \leq N$.<br><br>$\cdot$ for every edge, $1 \leq u, v \leq N$.<br><br>$\cdot$ for every query, $1 \leq u, c \leq N$.<br><br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For every test case, you should output &quot;<b>Case #x:</b>&quot; first,<br>where $x$ indicates the case number and counts from $1$.<br><br>Then for each query operation, output the number of different colors.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">1
5
1 2
1 3
2 5
2 4
1 2 3 4 5
6
1 1
0 4 2
1 1
1 2
0 2 3
1 2</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">Case #1:
