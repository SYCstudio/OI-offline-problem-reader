<tr><td align=center><h1 style='color:#1A5CC8'>The Diary of Math Teacher</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 2000/1000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 32768/32768 K (Java/Others)<br>Total Submission(s): 145&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 42<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>03/21/2010 SUN Sunny<br><br>“Hi, boys and girls ,today we will learn the “Deference” .”<br>“What is the “Deference”? Is that difficult? ”<br>“Boys , don’t worry, and you will know it immediately.”<br><br><br>“f(x) is an function, and x is an integer. Δf( x ) = f (x + 1 ) – f( x ) ”<br>“In general, Δ(n+1)f( x ) = Δ(n)f( x+1 ) - Δ(n)f( x )”<br>“In particular, Δ(0)f( x ) = f( x )，Δ(1)f( x ) = Δf (x )”<br><br><br>“Do you understand?”<br>“No!!!!!!!!!!!!”<br>“Does anyone understand?”<br>“Only you our teacher!”<br>“How dare you be so ironic. Since you all have learnt the Factorial. I will give you a problem of factorial and difference. Anyone who can’t solve it will be severely punished”<br><br><br>“ The problem is:<br>Given a nonnegative integer array {a[n]} , and each of its elements a[i] (1&lt;=i&lt;=n)is also known.<br>f(x)=∏{1&lt;=i&lt;=n}(x+a[i])<br>Your task is to calculate Δ(k)f(0)/k! ”<br>Damn it,really bad mood. Well, I'd better go to sleep now.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>Multiple test cases.<br>The first line of each test case will be 3 integers : n(1&lt;=n&lt;=1000) , k(0&lt;=k&lt;=n-1) , p(1&lt;=P&lt;=20000).<br>The following line will contain n integers a[i] (0&lt;=a[i]&lt;=20000).<br>Input ends with n=k=p=0.<br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For each test case ,pint the answer mod p in one separate line.<br>The last case n=k=p=0 does not need to be processed.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">5 3 20000
1 0 0 1 0
5 3 7
1 0 1 0 0
3 1 7777
1 1 0
3 0 94
0 1 1
7 4 10000
0 1 0 3 0 1 0
16 12 17595
4898 287 4879 598 5927 8 764 57 233 188 58 97 899 9876 47 323
0 0 0</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">38
3
4
0
748
4035


<div style='font-family:Times New Roman;font-size:14px;background-color:F4FBFF;border:#B7CBFF 1px dashed;padding:6px'><div style='font-family:Arial;font-weight:bold;color:#7CA9ED;border-bottom:#B7CBFF 1px dashed'><i>Hint</i></div>In the first test case:
f( x ) = ∏{1&lt;=i&lt;=5}(x + a[i]) = (x + a[1])(x + a[2])(x + a[3])(x + a[4])(x + a[5]) = (x + 1)*x*x*(x + 1)*x

So , f(0) = 0 , f(1) = 4 , f(2) = 72 , f(3) = 432
Δf(0) = 4 , Δf(1) = 68 , Δf(2) = 360
Δ(2)f(0) = 64 , Δ(2)f(1) = 292
Δ(3)f(0) = 228
3! = 1*2*3 = 6
228/6 = 38</div><i style='font-size:1px'> </i></div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Source</div> <div class=panel_content> <a href="/search.php?field=problem&key=2010+ACM-ICPC+Multi-University+Training+Contest%A3%A82%A3%A9%A1%AA%A1%AAHost+by+BUPT&source=1&searchmode=source"> 2010 ACM-ICPC Multi-University Training Contest（2）——Host by BUPT </a> </div> <div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Recommend</div> <div class=panel_content>zhouzeyong</div><div class=panel_bottom>&nbsp;</div><br><center style='font-size:15px;font-family:Arial;font-weight:bold;color:#1A5CC8'><a href='statistic.php?pid=3445'>Statistic</a>&nbsp;|&nbsp;<a href='submit.php?pid=3445'>Submit</a>&nbsp;|&nbsp;<a href="./discuss/problem/list.php?problemid=3445">Discuss</a> | <a href='note/note.php?pid=3445'>Note</a><br></td></tr><tr>
