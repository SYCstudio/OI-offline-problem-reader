<tr><td align=center><h1 style='color:#1A5CC8'>Map</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 3000/1500 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 262144/262144 K (Java/Others)<br>Total Submission(s): 645&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 213<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>   There are N pieces of fragments, and we’re going to select some of them to form a map. Some fragments can be selected into the map directly (named 1-level fragments), while each of the rest follows <b>exactly one</b> fragment as its prior fragment. If B follows A and A is an i-level fragment, then B is defined as an (i + 1) - level fragment, and B can be selected only if A has been selected before. Besides, a fragment is followed by at most one other fragment.<br><br>   Each fragment is assigned with a reliability. The reliability of the whole map is sum up by two parts:<br>   1.The sum of reliabilities of all selected fragments;<br>   2.If there are x<sub>i</sub> i-level fragments in total and we select y<sub>i</sub>(y<sub>i</sub> &gt; 1) fragments among them to form the map, the reliability would increase by <img style='max-width:100%;' src=../../data/images/C507-1001-1.jpg>,where S<sub>i</sub> denotes the sum of reliabilities of selected fragments in level i.<br><br>   Please figure out the expectation of reliability of the map under the condition that all valid selections are equiprobable.<br><br>   At least one fragment should be selected.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>   There are several test cases. The number of test cases T (T&lt;=10) occurs in the first line of input.<br><br>For each test case:<br>   The first line consists of two integers N, M (1&lt;=N&lt;=10000,0&lt;=M&lt;N), indicating the number of fragments and relationships respectively.<br>   The second line contains N integers, which describes the reliability of each fragment. The given reliability ranges from 1 to 100, inclusive.<br>   Each of the following M lines contains two integers A and B, which denotes<br>B follows A(0&lt;=A,B&lt;=N-1).<br><br>   You may assume that the number of 1-level fragments are no more than 10, and levels will not exceed 1000. No fragment will follow itself directly or indirectly.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>   For each test case, output the expectation with 3 decimal places.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">2
2 0
1 2
3 1
1 2 4
1 2</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">3.000
5.000
