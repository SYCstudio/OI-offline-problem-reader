<tr><td align=center><h1 style='color:#1A5CC8'>Math is Fun</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 3000/1500 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 65536/65536 K (Java/Others)<br>Total Submission(s): 50&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 10<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>A funny boy, XYZ introduced a simple math function called GLL for a set of integers$S=\{a_1,a_2,\cdots ,a_n \}$:<br>$$GLL(s) = GCD(S)*LCM(S)*LCM(S)$$<br><br>Here,$GCD(S) = GCD(a_1,a_2,\cdots ,a_n)$ means the greatest common divisor of integers $a_1,a_2,\cdots ,a_n$;$LCM(S) = LCM(a_1,a_2,\cdots ,a_n)$ means the least common multiple of integers $a_1,a_2,\cdots ,a_n$<br><br>For the singleton set, GCD and LCM will be the number only. For example, GCD of $S = \{x\}$ , will be $x$ only. Consider the LCM and GCD of an empty set as $0$ .<br><br>Now, he is interested in finding the sum of GLL values of all subsets for a given set $A$ , but he finds the problem very hard. Help him calculate the following:<br>$$Answer = \sum_{S\subset A}GLL(S)$$<br><br>As the answer can be very large, print it modulo 1000000009(10^9 + 9).</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>The first line contains T, the number of test cases. T test cases follow.<br><br>The first line of each test case contains N, the number of elements in $A$ ; the next line contains N space-separated positive integers.<br><br>$1 \leq T \leq 50$<br>$1 \leq N \leq 100$<br>Numbers in the array are in the range [1, 1000]<br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For each test case, output the answer in a newline.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">2
2
2 3
3
2 4 10</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">71
2904
