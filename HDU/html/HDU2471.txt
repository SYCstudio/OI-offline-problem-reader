<tr><td align=center><h1 style='color:#1A5CC8'>History of Languages</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 2000/5000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 32768/32768 K (Java/Others)<br>Total Submission(s): 767&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 240<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>We are examining two specific classes of languages (a possibly infinite set of<br>strings) in this problem. Fortunately (or maybe unfortunately), we are not given the strings contained in each language directly, rather we are given two deterministic finite automatons that describe such languages.<br><br>Your task here is simple: to determine if the languages described by the two automatons are the same.<br><center><img style='max-width:100%;' src=../../data/images/2471-1.jpg></center><br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>There are multiple test cases in the input file.<br>The first line of each test case is one integer, T ( 2 ÷ T ÷ 26), the size of the alphabet. In each test case, the description of automaton A comes before that of automaton B. The description of each automaton starts with one line containing N (1 ÷ N ÷ 2000 ), the number of states in the automaton, followed by N lines, each line of the format: F, X<sub>0</sub>, X<sub>1</sub>, ˇ, X<sub>T-1</sub>, <img style='max-width:100%;' src=../../data/images/2471-2.jpg>,-1 ÷ X<sub>i</sub> &lt; N). If F = 1, then the state is an accepting state; also, if X<sub>i</sub> = -1, it means that the state has no corresponding transition available for character i. The start state of both finite automatons will always be state 0.<br><br>Two successive test cases are separated by a blank line. A case with a single 0 indicates the end of the input file, and should not be processed by your program.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For each test case, please print “Yes” if the two languages described by the<br>automatons are equivalent; output “No” otherwise.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">2
3
1 -1 1
0 -1 2
0 -1 0
2
1 -1 1
0 -1 0
3
4
1 -1 -1 1
1 -1 -1 2
1 -1 -1 3
1 -1 -1 1
2
1 -1 -1 1
1 -1 -1 0
0</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">Case #1: No
