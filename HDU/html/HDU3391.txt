<tr><td align=center><h1 style='color:#1A5CC8'>Mahjong</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 2000/1000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 32768/32768 K (Java/Others)<br>Total Submission(s): 333&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 119<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>Mahjong is a wonderful game which origins from China (or maybe Korea because everything is Korean).<br>Mahjong is also a complex game. But the game we play here is rather easy. It just contains three suits: stones, bamboos and characters.<br>Stones consist of a number of circles. Each circle is said to represent can (筒, tóng) coins with a square hole in the middle.<br><center><img style='max-width:100%;' src=../../../data/images/3391-1.png></center> <br>Bamboos consist of a number of bamboo sticks. Each stick is said to represent a string (索, suǒ) that holds a hundred coins. Note that 1 Bamboo is an exception: it has a bird sitting on a bamboo, to prevent alteration.<br><center><img style='max-width:100%;' src=../../../data/images/3391-2.png></center> <br>Each character represents ten thousand (萬, wàn) coins.<br><center><img style='max-width:100%;' src=../../../data/images/3391-3.png></center> <br>A player wins the round by creating a standard mahjong hand, which consists of a certain number of melds (namely, four for 13-tile variations) and a pair. A meld is three tiles which are adjacent in one suit or all the same. A pair is two same tiles.<br>Now one player has three tiles in hand. Can you tell what more tile he need to win?<br>Note that the number of each tile in Mahjong is four. So if the number of one tile the player has is four, he cannot get this tile anymore.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>The first line contains one integer T indicating the number of test cases.<br>For each case, there are thirteen tiles in one line, separated by one space.<br>Each tile has two characters. The first character is ‘1’ to ‘9’ and the second is ‘s’ (for stone), ‘b’ (for bamboo), or ‘c’ (for character).</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For each case, output one line containing the case number and all the tiles he needs to win the round, separated by one space.<br>If he needs more than one tile, first output stone, then bamboo, finally character, all from 1 to 9.<br>If he cannot win after get any tile, output “None” instead.<br>Please follow the format of the sample output.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">5
1b 1b 2b 2b 3b 3b 5s 6s 7s 1c 1c 2c 2c
3s 4s 4s 5s 5s 5s 6s 6s 7s 9c 9c 4c 5c
1s 1s 1s 2s 3s 4s 5s 6s 7s 8s 9s 9s 9s
4b 5b 6b 7b 8b 8b 8b 2b 3b 4b 5s 6s 7s
4c 5c 6c 9b 9b 9b 9b 1s 2s 3s 2s 3s 4s</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">Case 1: 1c 2c
Case 2: 3c 6c
Case 3: 1s 2s 3s 4s 5s 6s 7s 8s 9s
Case 4: 1b 3b 4b 6b 7b 9b
Case 5: None

Hint
In case 3, if the player gets 1s, he can combine them into four melds (1s1s1s, 1s2s3s, 4s5s6s, 7s8s9s) and a pair (9s9s). And so it is with 2s to 9s.
2s: 1s1s1s, 3s4s5s, 6s7s8s, 9s9s9s, 2s2s
3s: 1s2s3s, 3s4s5s, 6s7s8s, 9s9s9s, 1s1s
4s: 1s1s1s, 2s3s4s, 4s5s6s, 7s8s9s, 9s9s
5s: 1s1s1s, 2s3s4s, 6s7s8s, 9s9s9s, 5s5s
6s: 1s2s3s, 4s5s6s, 6s7s8s, 9s9s9s, 1s1s
7s: 1s1s1s, 2s3s4s, 5s6s7s, 7s8s9s, 9s9s
8s: 1s1s1s, 2s3s4s, 5s6s7s, 9s9s9s, 8s8s
9s: 1s2s3s, 4s5s6s, 7s8s9s, 9s9s9s, 1s1s</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Author</div> <div class=panel_content>xay@whu</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Source</div> <div class=panel_content> <a href="/search.php?field=problem&key=The+5th+Guangting+Cup+Central+China+Invitational+Programming+Contest&source=1&searchmode=source"> The 5th Guangting Cup Central China Invitational Programming Contest </a> </div> <div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Recommend</div> <div class=panel_content>notonlysuccess</div><div class=panel_bottom>&nbsp;</div><br><center style='font-size:15px;font-family:Arial;font-weight:bold;color:#1A5CC8'><a href='statistic.php?pid=3391'>Statistic</a>&nbsp;|&nbsp;<a href='submit.php?pid=3391'>Submit</a>&nbsp;|&nbsp;<a href="./discuss/problem/list.php?problemid=3391">Discuss</a> | <a href='note/note.php?pid=3391'>Note</a><br></td></tr><tr>
