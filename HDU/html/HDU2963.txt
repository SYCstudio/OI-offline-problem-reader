<tr><td align=center><h1 style='color:#1A5CC8'>When</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 20000/10000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 32768/32768 K (Java/Others)<br>Total Submission(s): 109&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 3<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>&quot;When&quot; is an event driven language for machine control. It only has three statements: Set, Print, and the compound When clause. The (case insensitive) grammar is as follows<br><br>    PROGRAM := WHEN | PROGRAM WHEN<br>    WHEN := 'when ' EXPRESSION EOL STATEMENTS 'end when' EOL<br>    STATEMENTS := STATEMENT | STATEMENTS STATEMENT<br>    STATEMENT := PRINT | SET<br>    PRINT := 'print '  EXPRESSION_LIST EOL<br>    SET := 'set ' ASSIGNMENT_LIST EOL<br>    EXPRESSION_LIST := EXPRESSION | EXPRESSION_LIST ',' EXPRESSION<br>    ASSIGNMENT_LIST := ASSIGNMENT | ASSIGNMENT_LIST ',' ASSIGNMENT<br>    ASSIGNMENT := VARIABLE '=' EXPRESSION<br>    EXPRESSION := '(' EXPRESSION OP EXPRESSION ')' | VARIABLE | NUMBER <br>    OP :=  '&lt;' | '+' | '-' | 'and' | 'or' | 'xor'<br>    VARIABLE := '$' NOT_DOLLAR_STRING '$'<br>    NUMBER := DIGIT | NUMBER DIGIT<br>    DIGIT := '0' | .. | '9'<br>    NOT_DOLLAR_STRING := any sequence of printing characters (including blanks)<br>                         that does not contain a $ symbol.<br><br>In the above, any string enclosed in single quotes are to be treated literally. EOL is the end of line.<br><br>In words, a program consists of a list of when blocks, which themselves contain set and print statements. Case is ignored for key words and variable names. Spaces are allowed before or after any literal except inside a number. Spaces are allowed in variable names, and each non-empty sequence of spaces is treated as a single underscore, so the following refer to the same variable<br><br>     <br>    $Remote Switch#1$ <br>    $Remote_Switch#1$ <br>    $Remote   switch#1$ <br><br>All variable and literal values are integers between -1000000000 and 1000000000, inclusively. All variables are global and initially zero. The when programs you will be tested on will never have an EXPRESSION that evaluates to a value outside of this range. The logical operators evaluate to 0 for false and 1 for true, and treat any nonzero value as true.<br><br>Running the When program amounts to executing all the active when clauses until none are active. More specifically, the active list of when clauses is initially empty, then the following steps are repeated:<br><br>    * In the order they appear in the program, the conditions of all when clauses that are not currently active are evaluated. If true, the clause is added to the end of the active list, with its first statement marked as &quot;ready&quot;. Each active when clause has one &quot;ready&quot; statement.<br>    * If the active list is empty after this step, the program terminates.<br>    * The &quot;ready&quot; statement from the &quot;current&quot; when clause (initially the first clause in the active list) is executed.<br>    * The statement marked as &quot;ready&quot; is advanced, removing the when clause from the active list if this is the last statement in the &quot;current&quot; when clause.<br>    * The when clause marked as &quot;current&quot; is advanced, cycling to the beginning of the active list if the end is reached.<br><br>In other words, inactive when conditions are evaluated to determine if these clauses are added to the active list. Then one statement (set or print) is executed from the current active when clause. If this is the last statement in that clause, it is removed from the active list. One the next iteration, one statement is executed from the next active when clause, etc.<br><br>A set statement executes all the assignments concurrently, so that<br><br>    set $x$=$y$,$y$=$x$<br><br>swaps the values of $x$ and $y$. The same variable cannot appear twice on the left hand part of the same set statement (so set $x$=1,$x$=2 is illegal).<br><br>A print statement evaluates and prints the given expressions in the output, separated by commas and followed by a new line. So<br><br>    &nbsp;&nbsp;print 1,(2+3)<br><br>results in the line<br><br>    &nbsp;&nbsp;1,5<br><br>in the output. </div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>The input consists of several test cases separated by a blank line. Each of them is a single syntactically correct program. You may assume that the program will not execute more than 100000 set statements and 100000 print statements.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For each case, print the output produced by executing the given program. Print a blank line between the output of the cases.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">When ($Mr. Bill$&lt;5)
   Set $mr._bill$=($mr.  bill$+1),$Y$=($Y$+10)
End When
When ($mr. Bill$&lt;10)
  Set $MR. BILL$=($mr. bill$+1)
  Print $mr. bill$,$Y$
End When</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">3,20
6,40
7,40
8,40
9,40
10,40</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Source</div> <div class=panel_content> <a href="/search.php?field=problem&key=2008+Rocky+Mountain+Regional&source=1&searchmode=source"> 2008 Rocky Mountain Regional </a> </div> <div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Recommend</div> <div class=panel_content>gaojie</div><div class=panel_bottom>&nbsp;</div><br><center style='font-size:15px;font-family:Arial;font-weight:bold;color:#1A5CC8'><a href='statistic.php?pid=2963'>Statistic</a>&nbsp;|&nbsp;<a href='submit.php?pid=2963'>Submit</a>&nbsp;|&nbsp;<a href="./discuss/problem/list.php?problemid=2963">Discuss</a> | <a href='note/note.php?pid=2963'>Note</a><br></td></tr><tr>
