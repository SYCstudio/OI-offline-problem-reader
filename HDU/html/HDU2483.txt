Each element in the matrix is either “0” or “1”. A square is called magic square if it meets the following three conditions.<br>(1)&nbsp;&nbsp;The elements on the four borders are all “1”.<br>(2)&nbsp;&nbsp;Inside the square (excluding the elements on the borders), the number of “1”s and the number of “0”s are different at most by 1.<br>(3)&nbsp;&nbsp;The size of the square is at least 2 by 2.<br>Now given the matrix, please tell me how many magic square are there in the matrix.<br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>The input begins with a line containing an integer T, the number of test cases. <br>Each case begins with two integers R, C(1&lt;=R,C&lt;=300), representing the size of the matrix. The integers are separated by a single space.<br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For each case, output the number of magic square in a single line.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">3

5 5

1 1 0 1 1

2 2
1 1
1 1</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">3
