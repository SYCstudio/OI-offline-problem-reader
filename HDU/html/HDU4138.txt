<tr><td align=center><h1 style='color:#1A5CC8'>Cipher</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 2000/1000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 32768/32768 K (Java/Others)<br>Total Submission(s): 116&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 58<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>A spy is using a clever cipher technique to encode and decode messages.<br>The technique is very efficient as it enables him to send and receive any message, no matter how long it is with only three numbers!<br>After many great efforts, The Department of National Security managed to figure out how the technique works:<br>1. First of all, the only characters used are the space and lower case English letters. Each character is assigned an integer called the character volume, a space character has volume 1, ‘a’ has volume 2, ‘b’ has volume 3 and so on until ‘z’ which has volume 27. The volume of the whole message V is the sum of volumes of characters in the message.<br>2. A message consists of a number of words W, a word is a consecutive sequence of lower case letters, messages have no leading or trailing spaces and there is only one space between consecutive words.<br>3. For a certain V and W, let S be the lexicographically sorted set of messages that have volume V and consists of exactly W words. We can refer to a certain message using its one-based index I in S.<br>So when the spy wants to send a message M, he calculates its Volume V and the number of words it contains W and finds its index I in the corresponding set S (its index among all sorted messages of volume V with W words) and sends only  three numbers along: V, W and I!<br>The Department of National Security has done a great effort so far, yet they seek your help to decode messages sent by the spy! That is, given V, W and I you must decrypt the spy’s message or determine that there is no such one!</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>The first line of input contains an integer (1 &lt;= T &lt;= 200), the number of test cases.<br>Each data set consists of a line with three integers the volume of the message (1 &lt;= V &lt;= 75), the number of words (1 &lt;= W &lt;= 20), and the index of message (1 &lt;= I &lt;= 10<sup>18</sup>).</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For each test case, print one line containing the decrypted message or “Corrupted!” if there is no valid message corresponding to the given inputs.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">3
7 2 3
2 1 2
50 2 39098532022</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">Case #1: aa a
Case #2: Corrupted!
Case #3: big bang

<div style='font-family:Times New Roman;font-size:14px;background-color:F4FBFF;border:#B7CBFF 1px dashed;padding:6px'><div style='font-family:Arial;font-weight:bold;color:#7CA9ED;border-bottom:#B7CBFF 1px dashed'><i>Hint</i></div>
In the first test case, the sorted set of messages of volume 7 which contain 2 words is {“a aa”, “a c”, “aa a”,
“b b”, “c a”}, hence the third and required message is “aa a”.
In the second test case, the sorted set of messages of volume 2 which contain 1 word is {“a”}, the set has
only one message while index 2 is required, so something must have gone wrong!
