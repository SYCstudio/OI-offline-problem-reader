<tr><td align=center><h1 style='color:#1A5CC8'>Tree Cutting</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 4000/2000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 262144/131072 K (Java/Others)<br>Total Submission(s): 1252&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 487<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>Byteasar has a tree $T$ with $n$ vertices conveniently labeled with $1,2,...,n$. Each vertex of the tree has an integer value $v_i$.<br><br>The value of a non-empty tree $T$ is equal to $v_1\oplus v_2\oplus ...\oplus v_n$, where $\oplus$ denotes bitwise-xor.<br><br>Now for every integer $k$ from $[0,m)$, please calculate the number of non-empty subtree of $T$ which value are equal to $k$.<br><br>A subtree of $T$ is a subgraph of $T$ that is also a tree.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>The first line of the input contains an integer $T(1\leq T\leq10)$, denoting the number of test cases.<br><br>In each test case, the first line of the input contains two integers $n(n\leq 1000)$ and $m(1\leq m\leq 2^{10})$, denoting the size of the tree $T$ and the upper-bound of $v$.<br><br>The second line of the input contains $n$ integers $v_1,v_2,v_3,...,v_n(0\leq v_i &lt; m)$, denoting the value of each node.<br><br>Each of the following $n-1$ lines contains two integers $a_i,b_i$, denoting an edge between vertices $a_i$ and $b_i(1\leq a_i,b_i\leq n)$.<br><br>It is guaranteed that $m$ can be represent as $2^k$, where $k$ is a non-negative integer.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For each test case, print a line with $m$ integers, the $i$-th number denotes the number of non-empty subtree of $T$ which value are equal to $i$.<br><br>The answer is huge, so please module $10^9+7$.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">2
4 4
2 0 1 3
1 2
1 3
1 4
4 4
0 1 3 1
1 2
1 3
1 4</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">3 3 2 3
