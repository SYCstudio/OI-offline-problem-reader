<tr><td align=center><h1 style='color:#1A5CC8'>Emmet</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 2000/1000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 131072/131072 K (Java/Others)<br>Total Submission(s): 481&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 178<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>For every programmer, coding HTML &amp; CSS is a very boring thing.<br><br>For example, writing an html tag &lt;div&gt; with id &quot;div1&quot; and class &quot;col-md-3&quot;, you must write an html file like this:<br><br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;div id = &quot;div1&quot; class = &quot;col3&quot;&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;...<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;/div&gt;<br>Too much unnecessary coding!!!<br><br>For convenience, some Web programmer develop a vim plugin -- Emmet. By this tool, the programmer just need code &quot;div#div1.col3&quot; and then Emmet would transform it to &quot;&lt;div id = &quot;div1&quot; class = &quot;col3&quot;&gt;&lt;/div&gt;&quot;. It is very coollllll! Now you task is to write a program to perform this transformation.<br><br>Here are more details about you task:<br><br>1.Handle multilevel tag.<br>&nbsp;&nbsp;&quot;div&gt;p&gt;span&quot; means there are 3 tags and tag &lt;p&gt; is in the tag &quot;div&quot;, tag &lt;span&gt; is in the tag &quot;p&quot;.<br>&nbsp;&nbsp;So, the right answer is &quot;&lt;div&gt;&lt;p&gt;&lt;span&gt;&lt;/span&gt;&lt;/p&gt;&lt;/div&gt;&quot;<br><br>2.&nbsp;&nbsp;Every tag may have zero or one id and any amount of classes.<br>&nbsp;&nbsp;A string (only consisting of letters and digits) after '#' is an id name.<br>&nbsp;&nbsp;A string (only consisting of letters and digits) after '.' is a class name.<br>&nbsp;&nbsp;If a tag has id and classes at the same time, you must output the id first.<br>&nbsp;&nbsp;If a tag has more than one class, you must output them by the order according to the input.<br>&nbsp;&nbsp;For example<br>&nbsp;&nbsp;&quot;div.aa#bb.cc.ee&gt;p#g&gt;span.d&quot; =&gt;<br>&nbsp;&nbsp;&lt;div id=&quot;bb&quot; class=&quot;aa cc ee&quot;&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;p id=&quot;g&quot;&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&lt;span class=&quot;d&quot;&gt;&lt;/span&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;/p&gt;<br>&nbsp;&nbsp;&lt;/div&gt;&quot;<br><br>3.Handle parentheses.<br>&nbsp;&nbsp;Use parentheses to deal with sibling relation among tags!<br>&nbsp;&nbsp;For example<br>&nbsp;&nbsp;&lt;div id=&quot;bb&quot; class=&quot;aa cc ee&quot;&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;p id=&quot;g1&quot;&gt;&lt;span class=&quot;d1&quot;&gt;&lt;/span&gt;&lt;/p&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;p id=&quot;g2&quot;&gt;&lt;span class=&quot;d2&quot;&gt;&lt;/span&gt;&lt;/p&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;p id=&quot;g3&quot;&gt;&lt;span class=&quot;d3&quot;&gt;&lt;/span&gt;&lt;/p&gt;<br>&nbsp;&nbsp;&lt;/div&gt;<br>&nbsp;&nbsp;can be obtained by &quot;div.aa#bb.cc.ee&gt;(p#g1&gt;span.d1)(p#g2&gt;span.d2)(p#g3&gt;span.d3)&quot;<br>&nbsp;&nbsp;If the input string contains parentheses, the rightmost ‘)’ will be the last character of this string.<br><br>4.Handle symbol ‘*’<br>At the end of a tag, you may see a suffix &quot;*%d&quot;. It indicates that this tag would be repeated &quot;%d&quot; times.<br>&nbsp;&nbsp;For example<br>&nbsp;&nbsp;ul#id1&gt;li.classA*3&gt;p*2 =&gt;<br>&nbsp;&nbsp;&lt;ul id=&quot;id1&quot;&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;li class=&quot;classA&quot;&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&lt;p&gt;&lt;/p&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&lt;p&gt;&lt;/p&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;/li&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;li class=&quot;classA&quot;&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&lt;p&gt;&lt;/p&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&lt;p&gt;&lt;/p&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;/li&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;li class=&quot;classA&quot;&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&lt;p&gt;&lt;/p&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&lt;p&gt;&lt;/p&gt;<br>&nbsp;&nbsp;&nbsp;&nbsp;&lt;/li&gt;<br>&nbsp;&nbsp;&lt;/ul&gt;</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>   The first line of input contains an integer N (N&lt;=50), indicating the number of strings you need to transform.<br><br>   The following N lines, each consists of an input string. No string has more than 120 chars and the result would not have more than 1000 chars. Tag name, class name and id only contain English letters and digits. It is guaranteed that the input string is valid.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>   Output N lines each consisting of a string that is the result of the transformation. More details about the output format can be seen from the sample output. You should follow the output format strictly. No extra space or new line character is allowed in the output.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">4
div>p>span
div.aa#bb.cc.ee>p#g>span.d
div.aa#bb.cc.ee>(p#g1>span.d1)(p#g2>span.d2)(p#g3>span.d3)
ul#id1>li.classA*3>p*2</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">&lt;div>&lt;p>&lt;span>&lt;/span>&lt;/p>&lt;/div>
&lt;div id="bb" class="aa cc ee">&lt;p id="g">&lt;span class="d">&lt;/span>&lt;/p>&lt;/div>
&lt;div id="bb" class="aa cc ee">&lt;p id="g1">&lt;span class="d1">&lt;/span>&lt;/p>&lt;p id="g2">&lt;span class="d2">&lt;/span>&lt;/p>&lt;p id="g3">&lt;span class="d3">&lt;/span>&lt;/p>&lt;/div>
&lt;ul id="id1">&lt;li class="classA">&lt;p>&lt;/p>&lt;p>&lt;/p>&lt;/li>&lt;li class="classA">&lt;p>&lt;/p>&lt;p>&lt;/p>&lt;/li>&lt;li class="classA">&lt;p>&lt;/p>&lt;p>&lt;/p>&lt;/li>&lt;/ul></div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Author</div> <div class=panel_content>SYSU</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Source</div> <div class=panel_content> <a href="/search.php?field=problem&key=2014+Multi-University+Training+Contest+9&source=1&searchmode=source"> 2014 Multi-University Training Contest 9 </a> </div> <div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Recommend</div> <div class=panel_content></div><div class=panel_bottom>&nbsp;</div><br><center style='font-size:15px;font-family:Arial;font-weight:bold;color:#1A5CC8'><a href='statistic.php?pid=4964'>Statistic</a>&nbsp;|&nbsp;<a href='submit.php?pid=4964'>Submit</a>&nbsp;|&nbsp;<a href="./discuss/problem/list.php?problemid=4964">Discuss</a> | <a href='note/note.php?pid=4964'>Note</a><br></td></tr><tr>
