<tr><td align=center><h1 style='color:#1A5CC8'>EarthCup</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 2000/1000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 131072/131072 K (Java/Others)<br>Total Submission(s): 230&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 79<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>In the year of 2045,the soccer championship is gradually replaced by Earth Super Soccer Cup(We will call it EarthCup for short later).<br><br>For one EarthCup,there are $n(n\leq50000)$ soccer teams participated in. Every two teams will have a game. It means that every team will have $n-1$ games with all the other teams.<br><br>In order to make the result clear, it is ruled that if the two teams of a game have the same score when the game ends.Penalty shootout will last until there is a result.<br><br>In the EarthCup,every team has a Mark,and will score one point after winning a game and zero after losing a game. The team with the highest mark will be the champion.<br><br>In the year of 2333, somebody found that some teams had hired hackers to attack and modified the data of the EarthCup for many years.Maybe because of the great amount of the teams, this serious cheating behavior has not been found during hundreds of years.<br><br>To check whether the data was modified, they started to check the &quot;Mark Table&quot; in the past.<br><br>But because of the long ages, there were only the final Mark of each team reserved. No one remember the exact result of each game.Now they want to find out some &quot;Mark Table&quot; that must have been modified.<br><br>&quot;Must have been modified&quot; means we can not get this &quot;Mark Table&quot; by any exact result of each game according to the rules was given by.<br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>There is an positive integer $T(T\leq50)$ standing for the number of testcases.<br><br>For each testcase, there is an positive integer $n$ standing for the number of teams participate in EarthCup.<br><br>The followed n lines describe the &quot;Mark table&quot;. The $ith$ integer ${a}_{i}(0\leq {a}_{i} &lt; n)$ standing for the final Mark of the $ith$ team.<br><br><br><b>Hint:</b><br>For the first testcase:<br>One possible way is:<br>$Team1$ won $Team2$ and $Team3$,gained $2$ marks.$Team2$ won $Team3$ but lost the game with $Team1$,gained $1$ mark.$Team3$ lost both games and gained 0.<br><br>For the second testcase:<br>It's obviously impossible that all the teams would win all the games.So it must have been modified.<br><b> The Input ${a}_{i}$ is disordered.</b></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For each testcase, output &quot;The data have been tampered with!&quot;(without quotation) if it must have been modified,otherwise output &quot;It seems to have no problem.&quot;(without quotation).<br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">2
3
2
1
0
3
2
2
2</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">It seems to have no problem.
