<tr><td align=center><h1 style='color:#1A5CC8'>Convex Hull of Lattice Points</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 2000/1000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 32768/32768 K (Java/Others)<br>Total Submission(s): 402&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 246<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>A lattice point is a point with integer coordinates. A lattice polygon is a polygon with all vertices lattice points.<br><br>A polygon is convex if any line segment between two points of the polygon is inside (or on the boundary of) the polygon. Equivalently, the interior angle at each polygon vertex is less than 180 degrees.<br><br>For a set S, of lattice points, the convex hull is the smallest convex (lattice) polygon which contains all points of the set. (The vertices of the convex hull must be members of the set of lattice points). If all the points are on a single straight line, the convex hull will be a line segment (a degenerate polygon – see rightmost diagram below). In the diagrams below, the points of the set are indicated by solid dots, the vertices of the convex hull by X’s and the convex hull is drawn connecting the vertices.<br>Note that not all points on the convex hull polygon are vertices.<br><center><img style='max-width:100%;' src=../../data/images/C264-1007-1.jpg></center> <br>The vertices of a lattice polygon are in standard order if:<br>a) The first vertex is the one with the largest y value. If two vertices have the same y value, the one with the smaller x value is the first.<br>b) Vertices are given in clockwise order around the polygon. <br><br>Write a program that reads a set of lattice points and outputs the vertices of the convex hull of the points in standard order.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>The first line of input contains a single integer P, (1 ≤ P ≤ 1000), which is the number of data sets that follow. The first line of each data set contains the data set number, followed by a space, followed by a decimal integer giving the number of points N, (3 ≤ N ≤ 50), in the set. The remaining lines in the data set contain the points in the set, at most 5 points per line (the last line may have fewer). Each point consists of 2 space separated decimal integer values representing the x and y coordinates<br>respectively.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For each data set there are multiple lines of output. The first line contains a decimal integer giving the data set number followed by a single space, followed by a decimal integer giving the total number of vertices of the convex hull. The vertices of the convex hull follow, one per line in standard order. Each line contains the decimal integer x coordinate, a single space and the decimal integer y coordinate.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">4
