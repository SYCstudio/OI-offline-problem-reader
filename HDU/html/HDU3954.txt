<tr><td align=center><h1 style='color:#1A5CC8'>Level up</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 10000/3000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 32768/32768 K (Java/Others)<br>Total Submission(s): 4613&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 1296<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>Level up is the task of all online games. It's very boooooooooring. There is only level up in those games, except level up.<br>In a online game, there are N heroes numbered id from 1 to N, each begins with level 1 and 0 Experience. They need to kill monsters to get Exp and level up.<br><center><img style='max-width:100%;' src=http://acm.hdu.edu.cn/data/images/C361_1004_1.jpg></center><br>There are many waves of monsters, each wave, the heroes with id from li to ri will come to kill monsters and those hero with level k will get ei*k Exp. If one hero's Exp reach Needk then the hero level up to level k immediately.<br>After some waves, I will query the maximum Exp from li to ri.<br>Now giving the information of each wave and Needk, please tell me the answer of my query.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>The first line is a number T(1&lt;=T&lt;=30), represents the number of case. The next T blocks follow each indicates a case.<br>The first line of each case contains three integers N(1&lt;=N&lt;=10000), K(2&lt;=K&lt;=10) and QW(1&lt;=QW&lt;=10000)each represent hero number, the MAX level and querys/waves number.<br>Then a line with K -1 integers, Need2, Need3...Needk.(1 &lt;= Need2 &lt; Need3 &lt; ... &lt; Needk &lt;= 10000).<br>Then QW lines follow, each line start with 'W' contains three integers li ri ei (1&lt;=li&lt;=ri&lt;=N , 1&lt;=ei&lt;=10000); each line start with 'Q' contains two integers li ri (1&lt;=li&lt;=ri&lt;=N).</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For each case, output the number of case in first line.(as shown in the sample output)<br>For each query, output the maximum Exp from li to ri.<br>Output a black line after each case.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">2

Q 3 5</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">Case 1:
