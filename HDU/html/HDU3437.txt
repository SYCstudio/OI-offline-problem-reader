<tr><td align=center><h1 style='color:#1A5CC8'>Garden</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 2000/1000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 32768/32768 K (Java/Others)<br>Total Submission(s): 594&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 134<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>Every giantarum has its garden. So welcome to AekdyCoin’s garden! Here is the spray pond!<br><center><img style='max-width:100%;' src=../../data/images/3437-1.jpg></center> <br>The small blue point in the figure above is the headwater. The stream path is red. And there are some kind of dam-board, any of them has magic power!  When the water meets with the dam-board, something will happen!<br>Consider the headwater as one point in three-dimensional space and its coordinate is (0, 0, H). Then a velocity is given as the water’s horizontal velocity. You could see in the figure following:<br><br><center><img style='max-width:100%;' src=../../data/images/3437-2.jpg></center> <br><br>So if there is no dam-board cut off the stream path, the stream path will just as the yellow path in the figure above!( Air drag, friction will be ignore!)<br><br><center><img style='max-width:100%;' src=../../data/images/3437-3.jpg></center> <br><br>But if there exist some dam-board cut off the stream path<br> <br>So, if the dam-board cut off the stream, the Vertical velocity of the stream will become 0! And then depend on the magic power of the dam-board, the New_Vx ,New_Vy of New_V will change!<br><br>There are three kind of dam-board in AekdyCoin’s Garden.<br><br>(1)&nbsp;&nbsp;Type 0: the horizontal velocity vector rotate anticlockwise for some radian:<br><br><center><img style='max-width:100%;' src=../../data/images/3437-4.jpg></center> <br><br>(2)&nbsp;&nbsp;Type 1: The Vx will be modified!<br><br><center><img style='max-width:100%;' src=../../data/images/3437-5.jpg></center> <br><br>If the Vx is modified, we could know the horizontal velocity will be modified too!<br><br>(3)&nbsp;&nbsp;Type 2: The Vy will be modified!<br><br><center><img style='max-width:100%;' src=../../data/images/3437-6.jpg></center> <br><br>If the Vy is modified, we could know the horizontal velocity will be modified too!<br><br>Note: The modification happened at the moment when the stream touches the dam-board<br><br>Now AekdyCoin focus on a water-drop at the headwater and its mass is exactly M (kg).<br>Now Stupid AekdyCoin want you to write a program to tell him the time this water-drop need to drop onto the plane (z = 0). (In this problem, the acceleration due to gravity G is 9.18)<br><br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>In the first line there is an integer T, indicates the number of test cases. (T &lt;= 10)<br>In each case, the first line contains three real numbers Vx,Vy and H, indicates the  horizontal velocity and the headwater’s three dimensional coordinate is (0,0,H).<br>(-1000 &lt;= Vx,Vy &lt;= 1000, |H| &lt;= 1000)<br>Then a single line contains only one integer N, indicates the number of dam-board(0 &lt;= N &lt;= 10)<br>Then following N lines:<br>In each line, the first real number h is the height of the dam-board(|h| &lt;= 1000), and then an integer Ni indicates the coordinate of the Ni vertex of the dam-board(3&lt;=Ni&lt;=10). Then Ni coordinate (xi,yi) (-1000&lt;=xi,yi&lt;=1000,xi,yi are both integers)(Dam-board is always on a horizontal plane, so (xi,yi) indicates the three dimensional coordinate of the vertex is (xi,yi,h),and it is convex polygon.All the points will be given in clockwise or anticlockwise order!No two board share the same height!). Then one integer typ indicate the type of the dam-board, at last one real number k:<br>(1)&nbsp;&nbsp;If the typ = 0, then the k indicate the radian(|k| &lt;= 2.0 * PI,PI = 3.1415…)<br>(2)&nbsp;&nbsp;If typ = 1 or typ = 2 then we should change Vx or Vy into k! (|k| &lt;= 1000)<br><br><br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>Output “Case d: “first where d is the case number counted from one. Then output the time the water-drop need to drop onto the plane (z = 0), then output the answer rounded to 2 digits after the decimal point.<br>Otherwise if the water-drop could not drop onto the plane (z = 0) forever, just output “Forever!”.<br><br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">2

1 0 -5
0

1 0 18.36
1 
9.18 4 0 0 100 0 100 100 0 100 0 3.1415927</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">Case 1: Forever!
Case 2: 4.24
