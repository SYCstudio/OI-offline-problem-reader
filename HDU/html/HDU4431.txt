<tr><td align=center><h1 style='color:#1A5CC8'>Mahjong</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 4000/2000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 32768/32768 K (Java/Others)<br>Total Submission(s): 5600&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 1198<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>Japanese Mahjong is a four-player game. The game needs four people to sit around a desk and play with a set of Mahjong tiles. A set of Mahjong tiles contains four copies of the tiles described next:<br><center><img style='max-width:100%;' src=../../data/images/4431-1.jpg></center><br>One to nine Man, which we use 1m to 9m to represent;<br><center><img style='max-width:100%;' src=../../data/images/4431-2.jpg></center> <br>One to nine Sou, which we use 1s to 9s to represent;<br><center><img style='max-width:100%;' src=../../data/images/4431-3.jpg></center><br>One to nine Pin, which we use 1p to 9p to represent;<br><center><img style='max-width:100%;' src=../../data/images/4431-4.jpg></center><br>Character tiles, which are:Ton, Nan, Sei, Pei, Haku, Hatsu, Chun, which we use 1c to 7c to represent.<br><br>A winning state means a set of 14 tiles that normally contains a pair of same tiles (which we call &quot;eyes&quot;) and four melds. A meld is formed by either three same tiles(1m, 1m, 1m or 2c, 2c, 2c for example) or three continuous non-character tiles(1m, 2m, 3m or 5s, 6s, 7s for example).<br><br>However, there are two special winning states that are different with the description above, which are:<br>&quot;Chii Toitsu&quot;, which means 7 different pairs of tiles;<br>&quot;Kokushi Muso&quot;, which means a set of tiles that contains all these tiles: 1m, 9m, 1p, 9p, 1s, 9s and all 7 character tiles. And the rest tile should also be one of the 13 tiles above.<br><br>And the game starts with four players receiving 13 tiles. In each round every player must draw one tile from the deck one by one. If he reaches a winning state with these 14 tiles, he can say &quot;Tsu Mo&quot; and win the game. Otherwise he should discard one of his 14 tiles. And if the tile he throws out can form a winning state with the 13 tiles of any other player, the player can say &quot;Ron&quot; and win the game.<br><br>Now the question is, given the 13 tiles you have, does there exist any tiles that can form a winning state with your tiles?<br><br>(Notes: Some of the pictures and descriptions above come from Wikipedia.)<br></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>The input data begins with a integer T(1≤T≤20000). Next are T cases, each of which contains 13 tiles. The description of every tile is as above.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For each cases, if there actually exists some tiles that can form a winning state with the 13 tiles given, print the number first and then print all those tiles in order as the description order of tiles above. Otherwise print a line &quot;Nooten&quot;(without quotation marks).</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">2
1s 2s 3s 2c 2c 2c 2p 3p 5m 6m 7m 1p 1p
1p 1p 2p 3p 4s 5s 6s 7c 7c 3s 3s 2m 2m</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">2 1p 4p
Nooten</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Source</div> <div class=panel_content> <a href="/search.php?field=problem&key=2012+Asia+Tianjin+Regional+Contest&source=1&searchmode=source"> 2012 Asia Tianjin Regional Contest </a> </div> <div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Recommend</div> <div class=panel_content>zhoujiaqi2010</div><div class=panel_bottom>&nbsp;</div><br><center style='font-size:15px;font-family:Arial;font-weight:bold;color:#1A5CC8'><a href='statistic.php?pid=4431'>Statistic</a>&nbsp;|&nbsp;<a href='submit.php?pid=4431'>Submit</a>&nbsp;|&nbsp;<a href="./discuss/problem/list.php?problemid=4431">Discuss</a> | <a href='note/note.php?pid=4431'>Note</a><br></td></tr><tr>
