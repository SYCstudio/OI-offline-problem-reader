<tr><td align=center><h1 style='color:#1A5CC8'>Hun Gui Wei Company</h1><font><b><span style='font-family:Arial;font-size:12px;font-weight:bold;color:green'>Time Limit: 8000/4000 MS (Java/Others)&nbsp;&nbsp;&nbsp;&nbsp;Memory Limit: 131072/131072 K (Java/Others)<br>Total Submission(s): 519&nbsp;&nbsp;&nbsp;&nbsp;Accepted Submission(s): 139<br></span></b></font><br><br><div class=panel_title align=left>Problem Description</div> <div class=panel_content>My friend Hun Gui Wei (HGW) has started a company which is named Hun Gui Wei Company (HGWC). In HGWC, there are many staffs. Every staff has a certain salary, level and working age. Sometimes, HGW wants to do some queries. He wants to know the sum of salary of the staffs fit some conditions. For the large amount of staffs, artificial query is time-consuming. So he wants to hire you to write a program to help him. If you do a good job, he will pay you a generous remuneration.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Input</div> <div class=panel_content>Multi test cases (about 10).<br>The first line contains an integer n indicates there are n staffs in HGWC.<br>In the next n lines, each line describes the attributes of one staff.<br>Each line contains S, L, A indicate the salary, level and working age of one staff.<br>Next line an integer m comes which indicates there will be m queries.<br>Then next m lines, query dates will come<br>$\begin{matrix}<br>LL_0 &amp; HL_0 &amp; LA_0 &amp; HA_0 \\<br>LL_1 &amp; HL_1 &amp; LA_1 &amp; HA_1 \\<br>\vdots &amp; \vdots &amp; \vdots &amp; \vdots \\<br>LL_{m-1} &amp; HL_{m-1} &amp; LA_{m-1} &amp; HA_{m-1}<br>\end{matrix}$<br>HGW sets a variable k. In the beginning of each test case, k = 0. For the i-th query, he sets <br>$LL_i = LL_i + k, HL_i = HL_i - k, LA_i = LA_i + k, HA_i = HA_i - k$, then make k the answer of this query.<br><br>[Technical Specification]<br>All numbers in the input are integers.<br>$1 \leq n, m \leq 10^5$<br>$0 \leq S, L, A \leq 10^9$<br>$-10^{17} \leq LL_i, HL_i, LA_i, HA_i \leq 10^{17}$</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Output</div> <div class=panel_content>For i-th queryï¼Œoutput the sum of salary of the staffs whose level are between $LL_i$ and $HL_i$ while working age are between $LA_i$ and $HA_i$  in a single line.</div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Input</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">2
1 2 3
4 5 6
2
3 2 3 3
2 6 2 7</div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Sample Output</div><div class=panel_content><pre><div style="font-family:Courier New,Courier,monospace;">1
4
<div style='font-family:Times New Roman;font-size:14px;background-color:F4FBFF;border:#B7CBFF 1px dashed;padding:6px'><div style='font-family:Arial;font-weight:bold;color:#7CA9ED;border-bottom:#B7CBFF 1px dashed'><i>Hint</i></div>For this case, first we clear k i.e. set k to 0. For the first query we should find the staff whose level is between 2 and 3 while working age is exactly 3. There is only one staff fitting the conditions whose salary is 1. So the answer for the first query is 1. Then we set k to 1. For the second query we should update query parameters according to the statement. Thus after updating, the query parameters become 3 5 3 6. There is also only one staff fitting the condition whose salary is 4. So the answer for the second case is 4.</div><i style='font-size:1px'> </i></div></pre></div><div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Source</div> <div class=panel_content> <a href="/search.php?field=problem&key=BestCoder+Round+%2321&source=1&searchmode=source"> BestCoder Round #21 </a> </div> <div class=panel_bottom>&nbsp;</div><br><div class=panel_title align=left>Recommend</div> <div class=panel_content>heyang</div><div class=panel_bottom>&nbsp;</div><br><center style='font-size:15px;font-family:Arial;font-weight:bold;color:#1A5CC8'><a href='statistic.php?pid=5140'>Statistic</a>&nbsp;|&nbsp;<a href='submit.php?pid=5140'>Submit</a>&nbsp;|&nbsp;<a href="./discuss/problem/list.php?problemid=5140">Discuss</a> | <a href='note/note.php?pid=5140'>Note</a><br></td></tr><tr>
