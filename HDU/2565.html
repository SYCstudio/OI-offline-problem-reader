HDU2565. 放大的X

内存限制：2000/1000 MS (Java/Others) 时间限制：32768/32768 K (Java/Others)

题目描述

请你编程画一个放大的’X’。
如3*3的’X’应如下所示：

X X
 X
X X

5*5的’X’如下所示：
X   X
 X X
  X
 X X
X   X

输入格式

输入数据第一行是一个整数T，表示有T组测试数据；
接下来有T行，每行有一个正奇数n（3 <= n <= 79），表示放大的规格。

输出格式

对于每一个n打印一个规格为n * n放大的’X’；每组输出后面空一行。

样例

样例输入

2
3
5

样例输出

X X
 X
X X

X   X
 X X
  X
 X X
X   X
