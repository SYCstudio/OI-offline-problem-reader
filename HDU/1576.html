HDU1576. A/B

内存限制：1000/1000 MS (Java/Others) 时间限制：32768/32768 K (Java/Others)

题目描述

要求(A/B)%9973，但由于A很大，我们只给出n(n=A%9973)(我们给定的A必能被B整除，且gcd(B,9973) = 1)。

输入格式

数据的第一行是一个T，表示有T组数据。
每组数据有两个数n(0 <= n < 9973)和B(1 <= B <= 10^9)。

输出格式

对应每组数据输出(A/B)%9973。

样例

样例输入

2
1000 53
87 123456789

样例输出

7922
6060
