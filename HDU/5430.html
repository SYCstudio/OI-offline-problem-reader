HDU5430. Reflect

内存限制：2000/1000 MS (Java/Others) 时间限制：65536/65536 K (Java/Others)

题目描述

We send a light from one point on a mirror material circle,it reflects $N$ times and return the original point firstly.Your task is calcuate the number of schemes.


输入格式

First line contains a single integer $T (T \leq 10)$ which denotes the number of test cases.

For each test case, there is an positive integer $N (N \leq 10^{6})$.

输出格式

For each case, output the answer.

样例

样例输入

1
4

样例输出

4
