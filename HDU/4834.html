HDU4834. JZP Set

内存限制：10000/5000 MS (Java/Others) 时间限制：256000/256000 K (Java/Others)

题目描述

一个{1, ..., n}的子集S被称为JZP集，当且仅当对于任意S中的两个数x,y，若(x+y)/2为整数，那么(x+y)/2也属于S。
例如，n=3，S={1,3}不是JZP集，因为(1+3)/2=2不属于S。但是{1,2,3}的其他子集都属于S，所以n=3时有7个JZP集
给定n，求JZP集的个数。

输入格式

第一行为T，表示输入数据组数。
每组数据包含一行整数n。

限制条件
1<=T<=10^5
1<=n<=10^7

输出格式

对第i组数据，输出
Case #i:
然后输出JZP集的个数。

样例

样例输入

3
1
2
3

样例输出

Case #1:
2
Case #2:
4
Case #3:
7
