HDU3307. Description has only two Sentences

内存限制：3000/1000 MS (Java/Others) 时间限制：65536/32768 K (Java/Others)

题目描述

an = X*an-1 + Y and Y mod (X-1) = 0.
Your task is to calculate the smallest positive integer k that ak mod a0 = 0.

输入格式

Each line will contain only three integers X, Y, a0 ( 1 < X < 231, 0 <= Y < 263, 0 < a0 < 231).

输出格式

For each case, output the answer in one line, if there is no such k, output "Impossible!".

样例

样例输入

2 0 9

样例输出

1
