HDU5969. 最大的位或

内存限制：2000/1000 MS (Java/Others) 时间限制：65536/65536 K (Java/Others)

题目描述

B君和G君聊天的时候想到了如下的问题。
给定自然数l和r ,选取2个整数x,y满足l <= x <= y <= r ,使得x|y最大。
其中|表示按位或，即C、 C++、 Java中的|运算。

输入格式

包含至多10001组测试数据。
第一行有一个正整数，表示数据的组数。
接下来每一行表示一组数据，包含两个整数l,r。
保证 0 <= l <= r <= $10^{18}$。

输出格式

对于每组数据输出一行，表示最大的位或。

样例

样例输入

5
1 10
0 1
1023 1024
233 322
1000000000000000000 1000000000000000000

样例输出

15
1
2047
511
1000000000000000000
