HDU6216. A Cubic number and A Cubic Number

内存限制：1000/1000 MS (Java/Others) 时间限制：65535/32768 K (Java/Others)

题目描述

A cubic number is the result of using a whole number in a multiplication three times. For example, $3\times 3\times 3=27$ so $27$ is a cubic number. The first few cubic numbers are $1,8,27,64$ and $125$. Given an prime number $p$. Check that if $p$ is a difference of two cubic numbers.

输入格式

The first of input contains an integer $T~(1\le T\le 100)$ which is the total number of test cases.
For each test case, a line contains a prime number $p~(2\le p\le 10^{12})$.

输出格式

For each test case, output 'YES' if given $p$ is a difference of two cubic numbers, or 'NO' if not.
