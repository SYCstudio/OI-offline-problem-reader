HDU4767. Bell

内存限制：6000/3000 MS (Java/Others) 时间限制：32768/32768 K (Java/Others)

题目描述

What? MMM is learning Combinatorics!?
Looks like she is playing with the bell sequence now:
bell[n] = number of ways to partition of the set {1, 2, 3, ..., n}

e.g. n = 3:
{1, 2, 3}
{1} {2 3}
{1, 2} {3}
{1, 3} {2}
{1} {2} {3}
so, bell[3] = 5

MMM wonders how to compute the bell sequence efficiently.

输入格式

T -- number of cases
for each case:
　　n (1 <= n < 2^31)

输出格式

for each case:
　　bell[n] % 95041567

样例

样例输入

6
1
2
3
4
5
6

样例输出

1
2
5
15
52
203
