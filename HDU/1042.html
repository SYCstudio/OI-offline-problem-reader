HDU1042. N!

内存限制：10000/5000 MS (Java/Others) 时间限制：262144/262144 K (Java/Others)

题目描述

Given an integer N(0 ≤ N ≤ 10000), your task is to calculate N!

输入格式

One N in one line, process to the end of file.

输出格式

For each N, output N! in one line.

样例

样例输入

1
2
3

样例输出

1
2
6
